## 🚀 Quest Plugin v1.0.0 — First Official Release

Welcome to the first official release of **QuestPlugin** for DarkBot!
This plugin automates your daily quests in DarkOrbit — from killing NPCs to collecting resources — so you can focus on what matters.

---

### ✨ Core Features

- 🎯 **Automated Questing** — Handles KILL, COLLECT, CARGO, SALVAGE, PROXIMITY and TRAVEL quest objectives automatically for the active quest.
- 🗺️ **Smart Map Navigation** — Travels to the correct map required by the quest, including multi-hop portal routing.
- 🌑 **BlackLight Routing** — Prefers x-8 quest hubs to reduce travel time and danger in BlackLight zones.
- 📦 **COLLECT Quest Support** — Full support for box and resource collection quests. Manages cargo space intelligently and travels to the nearest refinery when full.
- ⚔️ **Mixed Quest Support** — Handles quests with combined KILL + COLLECT requirements in the correct sequence.
- 🔧 **Auto NPC Config** — Automatically configures which NPCs to attack or ignore based on the quest target, and restores original settings after completion.
- 🔥 **HeatMap Drawer** — Visual heatmaps and routing overlays for efficient NPC hunting across all maps.
- ⚠️ **Safety Warning** — Detects and warns about conflicting bot settings (e.g. `Only kill NPCs in Preferred area`) that could silently break quest progress.

---

### 💎 Premium Features (Donor Only)

- 🚀 **Rocket Auto Switch \[DONOR\]** — Automatically switches rocket types based on the current target (NPC or player) for maximum combat efficiency.
- 🤖 **Auto PET Gear \[DONOR\]** — Switches PET loadout dynamically:
  - Combat gear during NPC kill quests.
  - Looter gear during collect and cargo quests.
  - Enemy Locator mode with automatic ping tracking.
- 🥷 **Stealth \[DONOR\]** — Adds premium stealth behavior during route travel, coordinate quests, hotspot search, and random search movement.

---

### 📦 Installation

1. Download **`QuestPlugin-1.0.0.jar`** from the latest release assets.
2. Copy it into your DarkBot `plugins` folder.
3. Restart DarkBot.
4. In the `Plugins` tab, enable **Quest Module**.
5. If you want premium behavior, enable any donor feature you use:
   - `Rocket Auto Switch [DONOR]`
   - `Auto PET Gear [DONOR]`
   - `Stealth [DONOR]`
6. For premium features, follow the Discord link prompt shown in the plugin UI.

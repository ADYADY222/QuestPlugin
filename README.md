# Quest Plugin

Welcome to the **Quest Plugin** for DarkBot!

This plugin automates your daily quests in DarkOrbit - from killing NPCs to collecting resources.

## Download & Installation

Please go to the [Releases](https://github.com/ADYADY222/QuestPlugin/releases/latest) section on the right to download the plugin.

1. Download **`QuestPlugin-1.0.0.jar`** from the release assets.
2. Copy it into your DarkBot `plugins` folder.
3. Restart DarkBot and enable **Quest Module** in the Plugins tab.

**Do not use the green "Code -> Download ZIP" button.**

## Premium Features (Donor Only)
